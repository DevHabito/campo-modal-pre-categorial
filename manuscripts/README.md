# Manuscript Blueprints

This directory freezes the article-level architecture of the research
programme after the C1 and C2 novelty audits.

## Contents

- `GLOBAL_CLAIM_FREEZE.csv` and `.json` assign all 69 classified claims.
- `combinatorics/` contains the blueprint for the condensation-encoding paper.
- `foundational/` contains the blueprint for the informational-architecture
  paper.

## Status

- **Combinatorics:** scientifically blocked by the absence of a structural
  result for \(P_n\).
- **Foundational:** ready for a first integrated draft, subject to completion
  of classical attribution and reference checking.

## Freeze rule

A claim may move between main text, appendix, supplement, future work, or
repository-only status only through an explicit amendment to
`GLOBAL_CLAIM_FREEZE`.
