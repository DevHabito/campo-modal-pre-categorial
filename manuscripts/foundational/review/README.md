# Independent Review Package

This directory is designed for an external mathematical review of the
foundational manuscript without requiring the reviewer to reconstruct the
project's audit history.

## Suggested reading order

1. `ANONYMIZED_REVIEW_COPY.md`
2. `MATH_AUDIT_TARGETS.md`
3. `CRITICAL_CLAIM_CHECKLIST.csv`
4. `REVIEW_FORM.md`
5. `REVIEW_LOG_TEMPLATE.csv`

## Independence status

No external review has been completed or claimed in this package. The file
`INTERNAL_ADVERSARIAL_REVIEW.md` records an author-side stress test and must
not be counted as independent validation.

## Requested reviewer profile

At least one reviewer should have expertise in two or more of:

- probability and measure;
- functional equations or measurement theory;
- Markov-chain aggregation;
- statistical identifiability;
- coarse-graining or moment closure.

A second reader with foundations or philosophy-of-physics expertise would be
useful for the scope and relative-minimality language, but cannot replace the
mathematical review.
