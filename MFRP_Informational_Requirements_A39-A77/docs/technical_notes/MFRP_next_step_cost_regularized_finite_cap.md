# Cost-Regularized Finite-Cap Policy and Diminishing Returns

**Programme:** Modal Field Research Programme  
**Provisional audit:** A56  
**Author line:** Felipe Gianini Romero  
**Status:** exact decision theorem for the certified finite-cap implementation family; no empirical cost claim

## Technical abstract

A55 converted the compactified optimum

\[
\{2,\beta^\star,\infty\}\log2
\]

into the finite implementation family

\[
D_\Gamma
=
\{2,\beta^\star,\Gamma\}\log2,
\qquad
\Gamma\ge6,
\]

where

\[
\beta^\star
=
2.728401216547027876\ldots.
\]

The future-score risk is denoted

\[
Q_\Gamma
=
\mathcal R^Q(D_\Gamma).
\]

A55 proved that \(Q_\Gamma\) is generated by one exact primal–dual rational
branch for every \(\Gamma\ge6\), and that

\[
Q_\Gamma
=
Q_\infty
+
\kappa_Q2^{-\Gamma}
+
O(4^{-\Gamma}),
\]

with

\[
\kappa_Q
=
0.0020308153065398546459\ldots.
\]

The present audit proves two global properties of the finite implementation
frontier:

\[
\boxed{
Q_{\Gamma+1}<Q_\Gamma
}
\]

and

\[
\boxed{
G_{\Gamma+1}<G_\Gamma,
\qquad
G_\Gamma:=Q_\Gamma-Q_{\Gamma+1}.
}
\]

Thus increasing the finite cap always improves the risk, but each additional
unit produces a strictly smaller benefit than the previous one.

This permits a complete cost-regularized decision rule. Introduce a
nonnegative linear cost per cap unit,

\[
J_c(\Gamma)
=
Q_\Gamma+c\Gamma,
\qquad
c\ge0.
\]

The parameter \(c\) is measured in future-risk units per one-unit increase of
\(\Gamma\). It is not assigned a monetary or physical value by the theorem.

For \(c>0\), the optimal integer cap is finite. Define the break-even gains
\(G_\Gamma\). Then:

- if \(c>G_6\), the unique optimum is \(\Gamma=6\);
- if \(c=G_6\), \(\Gamma=6\) and \(7\) tie;
- for \(n\ge7\), if
  \[
  G_n<c<G_{n-1},
  \]
  the unique optimum is \(\Gamma=n\);
- if \(c=G_n\), the adjacent caps \(n\) and \(n+1\) tie;
- if \(c=0\), the risk infimum is the compactified limit
  \(\Gamma=\infty\).

The exact break-even schedule begins:

| Upgrade | Risk reduction \(G_\Gamma\) |
|---:|---:|
| \(6\to7\) | \(3.85431055963658\times10^{-5}\) |
| \(7\to8\) | \(1.32730434474054\times10^{-5}\) |
| \(8\to9\) | \(5.26376798709827\times10^{-6}\) |
| \(9\to10\) | \(2.30303201027043\times10^{-6}\) |
| \(10\to11\) | \(1.07100926048761\times10^{-6}\) |
| \(11\to12\) | \(5.15586126123988\times10^{-7}\) |
| \(12\to13\) | \(2.52839148900194\times10^{-7}\) |
| \(13\to14\) | \(1.25184290701080\times10^{-7}\) |
| \(14\to15\) | \(6.22837225700651\times10^{-8}\) |
| \(15\to16\) | \(3.10648052852303\times10^{-8}\) |
| \(16\to17\) | \(1.55131448499437\times10^{-8}\) |
| \(17\to18\) | \(7.75175875242441\times10^{-9}\) |
| \(18\to19\) | \(3.87467605500980\times10^{-9}\) |
| \(19\to20\) | \(1.93703720931998\times10^{-9}\) |

Asymptotically,

\[
\boxed{
G_\Gamma
=
\kappa_Q2^{-(\Gamma+1)}
+
O(4^{-\Gamma}).
}
\]

Therefore the selected cap under a small linear cost satisfies approximately

\[
\boxed{
\Gamma_c
\approx
\log_2
\left(
\frac{\kappa_Q}{2c}
\right).
}
\]

Every factor-of-ten reduction in the unit cost adds roughly

\[
\log_2 10
\approx3.32
\]

units to the preferred finite cap.

Representative exact decisions are:

| Unit cost \(c\) | Selected \(\Gamma\) |
|---:|---:|
| \(10^{-5}\) | 8 |
| \(10^{-6}\) | 11 |
| \(10^{-7}\) | 14 |
| \(10^{-8}\) | 17 |
| \(10^{-9}\) | 20 |

The result supplies a transparent stopping rule:

\[
\boxed{
\text{increase }\Gamma\text{ only while }G_\Gamma>c.
}
\]

## 1. Certified finite implementation family

The family is

\[
D_\Gamma
=
\{2,\beta^\star,\Gamma\}\log2,
\qquad
\Gamma\in\{6,7,8,\ldots\}.
\]

The first and second anchors remain fixed at the complete A54 optimum. Only
the finite approximation to the compactified endpoint is varied.

Set

\[
r=2^{-\Gamma}.
\]

A55 supplies an exact minimax ratio

\[
\rho(s^\star,r)
\]

for every

\[
0\le r\le\frac1{64}.
\]

The future-score risk is

\[
Q(r)
=
\frac12\log_2\rho(s^\star,r).
\]

Then

\[
Q_\Gamma=Q(2^{-\Gamma}).
\]

---

## 2. Strict risk improvement

The derivative of the exact ratio satisfies

\[
\frac{\partial\rho}{\partial r}>0
\]

on the full certified rectangle

\[
s\in
\left[
\frac3{20},
\frac{19}{125}
\right],
\qquad
r\in
\left[
0,\frac1{64}
\right].
\]

Because the logarithm is increasing,

\[
Q'(r)>0.
\]

But \(r=2^{-\Gamma}\) decreases when \(\Gamma\) increases. Therefore

\[
\boxed{
Q_{\Gamma+1}<Q_\Gamma
}
\]

for every integer \(\Gamma\ge6\).

Thus every increase of the cap improves the information risk.

---

## 3. Strict diminishing returns

Define the one-step improvement at continuous \(r\) by

\[
H(r)
=
Q(r)-Q(r/2).
\]

For integer caps,

\[
G_\Gamma
=
H(2^{-\Gamma}).
\]

Its derivative has the same sign as the rational expression

\[
F(s,r)
=
\frac{\partial_r\rho(s,r)}{\rho(s,r)}
-
\frac12
\frac{
\partial_r\rho(s,r/2)
}{
\rho(s,r/2)
}.
\]

An exact Bernstein sign certificate proves

\[
F(s,r)>0
\]

on the full A55 rectangle. Therefore

\[
H'(r)>0.
\]

Since

\[
2^{-(\Gamma+1)}
<
2^{-\Gamma},
\]

\[
H(2^{-(\Gamma+1)})
<
H(2^{-\Gamma}).
\]

Hence

\[
\boxed{
G_{\Gamma+1}<G_\Gamma.
}
\]

The finite-cap frontier has globally diminishing marginal returns.

---

## 4. Pareto frontier

The two criteria are:

1. lower future-score risk;
2. lower finite-cap complexity.

For

\[
\Gamma_1<\Gamma_2,
\]

we have

\[
Q_{\Gamma_2}<Q_{\Gamma_1},
\]

but

\[
\Gamma_2>\Gamma_1.
\]

Therefore neither point dominates the other. Every integer

\[
\Gamma\ge6
\]

lies on the risk–complexity Pareto frontier of the certified implementation
family.

A cost or resource preference is required to select one point from this
frontier.

---

## 5. Linear cost theorem

Let

\[
J_c(\Gamma)
=
Q_\Gamma+c\Gamma,
\qquad
c>0.
\]

The forward difference is

\[
J_c(\Gamma+1)-J_c(\Gamma)
=
c-G_\Gamma.
\]

Because \(G_\Gamma\) is strictly decreasing, these differences change sign at
most once.

### Theorem 5.1

For every \(c>0\), the integer minimizer is characterized as follows:

\[
\Gamma_c=6
\quad\text{if}\quad
c>G_6.
\]

For \(n\ge7\),

\[
\Gamma_c=n
\quad\text{if}\quad
G_n<c<G_{n-1}.
\]

At a threshold \(c=G_n\), the two adjacent caps

\[
n
\quad\text{and}\quad
n+1
\]

tie.

Thus the policy is complete and monotone: smaller unit cost never produces a
smaller selected cap.

---

## 6. Asymptotic selection law

From A55,

\[
Q_\Gamma
=
Q_\infty
+
\kappa_Q2^{-\Gamma}
+
O(4^{-\Gamma}).
\]

Therefore

\[
\begin{aligned}
G_\Gamma
&=
Q_\Gamma-Q_{\Gamma+1}\\
&=
\kappa_Q
\left(
2^{-\Gamma}-2^{-(\Gamma+1)}
\right)
+
O(4^{-\Gamma})\\
&=
\frac{\kappa_Q}{2}2^{-\Gamma}
+
O(4^{-\Gamma}).
\end{aligned}
\]

The break-even condition

\[
c\approx G_\Gamma
\]

gives

\[
\Gamma_c
\approx
\log_2
\left(
\frac{\kappa_Q}{2c}
\right).
\]

This is not used as the exact decision rule. It explains the scale of the
exact threshold table.

---

## 7. Interpretation for implementation

The cap \(\Gamma\) controls how closely the finite observation approximates
the compactified endpoint.

A larger cap gives a better approximation, but the improvement per added unit
shrinks rapidly.

The quantity

\[
G_\Gamma
\]

is the exact informational value of the next unit of cap. Therefore:

- if the next unit costs less than \(G_\Gamma\), increase the cap;
- if it costs more than \(G_\Gamma\), stop;
- if it costs exactly \(G_\Gamma\), either adjacent cap is equally good.

This converts the vague phrase “large enough \(\Gamma\)” into a reproducible
decision policy.

## 8. Logical status

### Established

1. The finite-cap risk strictly decreases for every integer
   \(\Gamma\ge6\).
2. The one-step risk reductions strictly decrease.
3. Every cap lies on the risk–complexity Pareto frontier.
4. The linear-cost problem has a complete threshold solution.
5. Exact break-even values are supplied.
6. The selected cap grows logarithmically as the unit cost decreases.
7. The policy uses the exact A55 minimax branch, not a fitted exponential
   curve.

### Not established

1. The cost parameter \(c\) is not estimated from a real instrument.
2. The theorem optimizes the fixed family
   \(\{2,\beta^\star,\Gamma\}\), not all possible finite-cap choices of
   \(\beta\).
3. Nonlinear or discontinuous cost functions are not optimized here.
4. No monetary, temporal, energetic, or hardware unit is assigned.
5. No empirical error profile is included.

---

## 9. Next rigorous target

The cost-free Pareto frontier and the linear-cost policy are now closed.

The next extension should allow the observation error itself to depend on the
cap:

\[
\varepsilon=\varepsilon(\Gamma).
\]

That is the realistic case in which more extreme observations may become
noisier rather than merely more expensive. It will determine whether the
optimal cap remains finite even when the explicit cost \(c\) is zero.
