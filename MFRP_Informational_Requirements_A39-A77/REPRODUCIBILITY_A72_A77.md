# Reproducibility and Scope — A72–A77

The package distinguishes four layers:

1. **Exact rational/algebraic certificates:** phase branches, Cramer determinants, Bernstein coefficient signs, primal-dual equality, and KKT signs.
2. **Finite exhaustive catalogues:** explicitly declared integer anchor catalogues and one-pivot neighborhoods.
3. **Numerical discovery artifacts:** dense scans used to discover candidate phase signatures before exact certification.
4. **Superseded exploratory artifacts:** preserved only in the raw-provenance archive.

No numerical discovery scan is silently promoted to an exact global theorem. Every note states the interval, support sizes, contact family, or neighborhood to which its conclusion applies.
